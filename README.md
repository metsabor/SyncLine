**🔷 SyncLine — Быстрый и стильный мессенджер**

*SyncLine* — десктопный мессенджер с современным дизайном в стиле *Glassmorphism*, проект создан для быстрого обмена сообщениями, файлами, создания каналов и групп, а также для голосового общения. Всё это — с уникальной фишкой: *«монеткой»* за ежедневное общение.

**🚀 Особенности**

- **Авторизация через Telegram** — вход по username, код приходит в *Telegram*.
- **Каналы и группы** — создавайте публичные или приватные сообщества.
- **Голосовые каналы** — общайтесь в реальном времени.
- **Одноразовые файлы** — отправляйте фото/видео, которые сгорают после просмотра.
- **Реакции, эмодзи, гифки** — полный набор эмоций.
- **Монетка** — за ежедневное общение с пользователем вы получаете монетки, которые растут в цене *(аналог огонька в TikTok, но с уникальной механикой)*.
- **Музыкальный плеер** — встроенный плеер с поддержкой локальных *MP3-файлов*.
- **Профиль** — дата рождения, закреплённые каналы, статус.
- **Тёмная тема** — стильный интерфейс с эффектами стекла *(Glassmorphism)*.
- **Портативная версия** — не требует установки, работает из папки.
